# Asset: High-Fidelity Deletion Modal

## Logic Summary
This pattern provides a high-contrast, deliberate confirmation flow using a custom-styled modal. It preferred over browser popups or inline text changes for critical destructive actions.

## UI Style (Clean White Design)
The design prioritizes clarity and high-contrast danger signaling.
- **Background**: Light/Blurry overlay (`backdrop-blur-sm`).
- **Surface**: Sharp white card with large rounded corners (`rounded-[24px]`).
- **Primary Action**: Bold Red button (`bg-[#FF3B30]`).
- **Visual Cues**: Title in bold, warning text in grey, and a footer reminder "This action is permanent."
